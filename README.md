# SmartStone V3 Elite - Tamamlanmış Proje Rehberi 💎

Bu makine, 35x35cm alanda SS4-SS16 taşları 0.1mm hassasiyetle dizen çift kafalı (IDEX) bir CNC sistemidir.

## 📁 Dosya Yapısı:
- **/CODE**: Arduino Mega için özelleştirilmiş hareket ve vakum kodları.
- **/ELECTRONICS**: Pin bağlantıları ve tam malzeme listesi (BOM).
- **/MECHANICAL**: Montaj adımları ve 8'li döner hazne ölçüleri.

## 🛠 Teknik Özellikler:
- **Hız:** 40 taş/dakika.
- **Sürücüler:** TMC2209 (Sessiz mod).
- **Hassasiyet:** 0.1mm taş aralığı.
